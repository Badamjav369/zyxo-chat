# 🔐 ZYXO-CHAT

ZYXO-CHAT нь **End-to-End Encryption (E2EE)** технологид суурилсан real-time чат систем.  
Энэхүү систем нь хэрэглэгчдийн мессежийг шифрлэж серверээр дамжуулдаг. Мессежийг зөвхөн илгээгч, хүлээн авагч тал унших боломжтой.

---

## 🛠 Ашигласан технологиуд

- 🟢 Node.js
- ⚡ Express.js
- 🔌 Socket.IO
- 🗄 MySQL
- 🔑 JWT Authentication
- 🛡 bcrypt
- 🔐 CryptoJS

---

## ✨ Үндсэн боломжууд

- 💬 Real-time Chat
- 🔒 End-to-End Encryption
- 👤 User Authentication
- 🟢 Online Status
- 🛡 Secure Messaging
